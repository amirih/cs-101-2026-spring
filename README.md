# CS 101: Programming for Future Advanced Engineers

This repository contains course materials for **CS 101: Programming for Future Advanced Engineers** for the **Spring 2026** semester.

## Repository Structure

```text
code/
  python/    Example Python programs and practice code
  inclass/  Weekly class problem solutions and related examples

docs/     Lecture notes, slides, and other course materials
```

## Updates

The `inclass/` directory will be updated if we make any changes to the in-class problem solutions. The `code/python/` directory will be updated with new example programs and practice code as we cover new topics in the course. I will also update the `docs/` directory with any new lecture notes or materials as they become available.
